# Web Development Fundamentals Practical Experiments

This repository contains the complete set of **Web Development Fundamentals practical experiments** as prescribed in the syllabus.  
The experiments cover **HTML, CSS, JavaScript, DOM manipulation, and basic Git & deployment concepts**, organized unit-wise for academic submission.

---

## Tools and Technologies Used

- **Languages:** HTML5, CSS3, JavaScript  
- **Code Editor:** Visual Studio Code  
- **Browser:** Google Chrome / Mozilla Firefox  
- **Version Control:** Git  
- **Deployment Platforms:** GitHub Pages / Netlify  

---

## File Naming Convention

Each experiment file follows the format:

task<number>.html

### Examples

task1.html  
task2.html  
task10.html  

Some tasks may include embedded CSS and JavaScript within the same HTML file.

---

## List of Experiments

### UNIT–1: HTML Basics

1. Create a basic HTML web page using headings, paragraphs, and text formatting tags  
2. Design a webpage demonstrating HTML links, images, and lists (ordered, unordered, nested)  
3. Create a table-based layout with merged cells, alignment, and caption using HTML  
4. Design a webpage with an HTML form having text input, radio buttons, checkboxes, select menus, and submit/reset buttons  
5. Use semantic elements (article, section, nav, aside, footer) to build a structured web page  

---

### UNIT–2: CSS

6. Apply Inline, Internal, and External CSS styles to HTML elements  
7. Style a web page using advanced CSS: Box Model, background images, borders, margins, and padding  
8. Demonstrate the use of different CSS selectors (class, id, descendant, group, universal)  
9. Create a responsive web page using media queries and mobile-first design with flexible units  
10. Use div and span elements for layout and apply styling with CSS  

---

### UNIT–3: JavaScript Basics

11. Create a JavaScript-enabled web page that performs basic arithmetic operations using user input  
12. Write JavaScript to demonstrate conditional statements and looping (for, while, do-while)  
13. Create and invoke user-defined functions in JavaScript; use var, let, and const for scope demonstration  
14. Handle HTML form validation using JavaScript events like onsubmit and oninput, and use alert for feedback  

---

### UNIT–4: DOM and Advanced JavaScript

15. Demonstrate DOM manipulation using getElementById, querySelector, and innerHTML  
16. Show or hide HTML elements using JavaScript and toggle CSS classes dynamically  
17. Add interactivity using event listeners for mouse and keyboard events  
18. Create and manipulate elements using JavaScript: append, remove, or modify child nodes  
19. Create a simple drawing using the canvas element  

---

### UNIT–5: Git and Deployment

20. Initialize a Git repository, commit changes, and push to GitHub; deploy a personal portfolio site on GitHub Pages or Netlify  

---

## Objective

The objective of this practical file is to:

- Understand the structure of web pages using HTML  
- Apply styling using CSS  
- Implement interactivity using JavaScript  
- Learn DOM manipulation techniques  
- Gain hands-on experience with Git and web deployment
